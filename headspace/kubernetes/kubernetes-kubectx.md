# KUBERNETES KUBECTX

## Resources
- [Kubernetes kubectx](https://github.com/ahmetb/kubectx)

## Features

- quickly change context
- quickly change namespace

## Install

`brew install kubectx`

## Switch contexts

`kubectx cluster1`

## Switch to previous context

`kubectx -`

## Switch namespace

`kubens kube-system`

## Switch to previous namespace

`kubens -`
