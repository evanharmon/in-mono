# KUBERNETES EXAMS VIM

## Settings to change

```sh
echo 'set expandtab' >> ~/.vimrc
echo 'set tabstop=2' >> ~/.vimrc
echo 'set softtabstop=2' >> ~/.vimrc
echo 'set shiftwidth=2' >> ~/.vimrc
```

or the short versions
```sh
echo 'set expandtab' >> ~/.vimrc
echo 'set ts=2' >> ~/.vimrc
echo 'set sts=2' >> ~/.vimrc
echo 'set sw=2' >> ~/.vimrc
```
