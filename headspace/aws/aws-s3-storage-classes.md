## Storage Classes

- S3 Standard
- S3-Infrequent Access (IA)
- S3 One Zone-Infrequent Access
- S3 Intelligent-Tiering
- S3 Glacier Instant Retrieval
- S3 Glacier Flexible Retrieval
- S3 Glacier Deep Archive
- Reduced Redundancy (no longer recommended)
