# RUST CRATES CARGO-WATCH

## Resources

- [Crates cargo-watch](https://crates.io/crates/cargo-watch)

## Features

- watches for code changes and restarts like `nodemon`

## Global Install

```console
cargo install cargo-watch
```

## Use

```console
cargo-watch -x run
```

## Watch cargo test

```console
cargo-watch -x test
```
