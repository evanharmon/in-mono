# RUST CRATE CARGO-DOCS

## Resources

- [Rust crate cargo-docs](https://crates.io/crates/cargo-docs)

## Install

```console
cargo install cargo-docs
```

## Open docs locally in a browser

```console
cargo docs -ro
```

## Live reload on doc changes

```console
cargo docs -row
```

## Serve just rust docs

```console
cargo docs -bro
```
