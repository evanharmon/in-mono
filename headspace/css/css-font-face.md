# CSS FONT-FACE

## Include Diff Weights Of Google Font

```css
@font-face {
  font-family: 'Roboto';
  src: url('/PS.UI/assets/fonts/Roboto-Thin.ttf') format('truetype');
  font-weight: 100;
  font-style: normal;
}
```

```css
@font-face {
  font-family: 'Roboto';
  src: url('/PS.UI/assets/fonts/Roboto-ThinItalic.ttf') format('truetype');
  font-weight: 100;
  font-style: italic;
}
```
