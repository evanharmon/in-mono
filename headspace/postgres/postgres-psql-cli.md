# POSTGRES

## Resources

- [Postgres PSQL CLI Docs](https://www.postgresql.org/docs/current/app-psql.html)

## Login with username

```console
psql --username postgres
```

## List Roles

```console
\du
```

## List Databases

```console
\l
```

## Quit

```console
\q
```

## Connect to database

```console
\c pagila
```

## Show config file location

```console
SHOW config_file
```

## List catalog schemas

```console
\dn
```
