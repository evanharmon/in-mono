# PYTHON CONDITIONALS

## If, elseif, else

```python
if x == 5:
     # expression
elif x == 6:
     # expression
else:
     # expression
```

## Multiple checks

```python
if x == 5 and y == 10:
     # expression
```

```python
if temp > 30:
     # expression
if 20 <= temp <= 30:
     # expression
else:
     # expression
```
