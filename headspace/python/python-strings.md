# PYTHON STRINGS

## Convert

```python
x = 10
str(10)
```

## Concat

```python
x = 'hello'
y = 'world'
print x + y
```

## Join strings

```python
a = "hello"
b = "world"
c = a.join(b)
```

## Repeat string

`print x + 4`

## Split string to array

`sen = list("hello World")`

## Split String By Separator

```python
keyPath = "my/file/path.txt"
print(keyPath.split("/"))
```

## Reverse string

```python
string = "hello world"
print string[::-1]
```

## check if digit

```python
string = "5"
string.isdigit()
```

## check if number

```python
num = "5"
num.isnumeric()
```

## Compare strings

### returns index

```python
string = "hello world"
lookfor = "hello"
string.find(lookfor)
```

## Title Case

`string.title()`

## Check Uppercase

`string.isupper()`

## Switch Case

`string.swapcase()`

## Check if alphabet letter

`string.isalpha()`
