# LINUX PASSWD

## Features
- changes passwords
- requires root

## Flags
- `-e` expiry date
- `-c` custom comments
- `-d` custom home directory
- `-g` gid
- `-u` uid
- `-s` login shell
- `-G` create user with multiple secondary groups

## Command

### Change password for a user
`passwd evan`

### Change own password
`passwd`
